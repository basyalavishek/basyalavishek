## 🚀 Featured Projects

### 🌟 [AI Detector and Humanizer](https://github.com/basyalavishek/AI-detector-and-humanizer)
*An adversarial AI pipeline engineered to detect and bypass automated AI-text classifiers via intelligent, iterative rewriting.*
*   **Core Engineering:** Developed a stateful, cyclical agentic framework using **LangGraph** to implement an automated critique-and-refine evaluation loop.
*   **Key Features:** Integrated local evaluation modules (`ai-text-detector-model`, `AI_Detect_Model`), designed conditional routing nodes based on detection confidence thresholds, and managed environments efficiently using `uv`.
*   **Tech Stack:** Python, FastAPI, LangGraph, Hugging Face Transformers, Local LLMs/Detectors.

### 🧠 [RAG-Chat Ecosystem](https://github.com/basyalavishek/RAG-Chat)
*A robust, context-aware conversational backend engineered to minimize hallucinations using document-grounded retrieval pipelines.*
*   **Core Engineering:** Orchestrated modular Retrieval-Augmented Generation workflows using **LangChain** to handle document parsing, semantic chunking, and contextual query routing.
*   **Key Features:** Efficient vector embeddings management, dynamic history tracking, and clear split-logic middleware for optimized backend text processing.
*   **Tech Stack:** Python, LangChain, Vector Store/ChromaDB, Next.js, FastAPI.
