## 🚀 Featured Projects

### 🌟 [AI Detector and Humanizer](https://github.com/basyalavishek/AI-detector-and-humanizer)
*A full-stack adversarial AI system that detects AI text with sentence-level precision and dynamically rewrites flagged content.*
* **Surgical Loop Architecture:** Leveraged **LangGraph** to build a 3-iteration `detect → rewrite → re-detect` loop that only alters flagged text while preserving original document layout.
* **Hybrid Execution:** Orchestrated a fine-tuned local DistilBERT classifier for detection alongside an LLM layer (Ollama/Groq) for contextual humanization.
* **Tech Stack:** Python, FastAPI, LangGraph, Hugging Face (DistilBERT), Next.js, TypeScript.

### 🧠 [RAG-Chat Ecosystem](https://github.com/basyalavishek/RAG-Chat)
*A full-stack app that lets users upload documents into isolated chat sessions and ask questions about their content.*
* **Smart Chat Sessions:** Built a system where each chat has its own private document storage, vector search index, and memory history.
* **Smooth Streaming:** Allowed answers to stream back word-by-word in real time while using an LLM to rewrite user questions based on past chat context.
* **Tech Stack:** Python, FastAPI, LangChain, ChromaDB, SQLite, React, TypeScript.
